
# 🚀 AIventory

## 📌 Project Overview

AIventory is a full-stack inventory management system built using React, FastAPI, and PostgreSQL. It enables efficient product management, real-time stock tracking, and scalable backend operations, with future scope for AI-based insights such as demand prediction and smart analytics.

---

## 🛠 Tech Stack

* Frontend: React.js
* Backend: FastAPI
* Database: PostgreSQL
* ORM: SQLAlchemy
* Data Validation: Pydantic

---

## ✨ Features

* Perform CRUD operations on products  
* Real-time inventory tracking  
* RESTful API integration  
* Scalable backend architecture  
* Data validation using Pydantic  
* Clean and modular code structure  
* Future scope: AI-based analytics and demand prediction  

---

## 🔄 Project Flow

Frontend (React) → API Calls → Backend (FastAPI) → Database (PostgreSQL)

---

## ⚙️ How to Run

### Backend

```bash
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
